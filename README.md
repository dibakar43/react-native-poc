# react-native-poc


1. Download and install node.js
* https://nodejs.org/en/download/

2. Run the following command in command prompt
* npm install -g create-react-native-app
React native app doesn’t support npm 5 till now.
So, if you face this kind of exception. Please run following command to downgrade npm.
* npm install -g npm@4

3. Then run the following commands to create a new React Native project called "AwesomeProject"	
* create-react-native-app AwesomeProject

4. Go inside the AwesomeProject by doing the change directory and run the following command to start development server , and print a QR code in your terminal
* npm start
5. Install Expo in your android device from play store. Then, scan the QR code to test the app.
6. Register yourself in git.
7. Download git desktop in you system and sign in with the same account.
8. Download visual Studio code. We are going to use VS code as IDE.
